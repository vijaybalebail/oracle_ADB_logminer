1. Unlocked and configured GGADMIN account

Key Findings:
- Must use GGADMIN user for LogMiner in ADB Serverless (undocumented)
- SUPPLEMENTAL LOGGING must be enabled on tables (ALL COLUMNS recommended)
- DICT_FROM_ONLINE_CATALOG option is REQUIRED for table name decoding
- Time-based ranges (STARTTIME/ENDTIME) work better than SCN ranges
- LogMiner captures all DML with complete SQL_REDO and SQL_UNDO
- LogMiner session terminates on CONNECT - must restart to query
- Only ONE LogMiner session allowed per ADB instance
- Archived logs retained for 7 days in ADB Serverless
- V$LOGMNR_CONTENTS queries can be slow - query once and filter results
